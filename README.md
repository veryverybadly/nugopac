##  Как войти (Установка)

### Windows (Confirmed)
1. **Распаковать в тёплое место.**
2. **Запустить `NugopacEditor.exe`.**
3. **Кайфовать от зависшего окна.**

### Linux (Wine)
1. **хз не шарю.**
2. **Запустить `NugopacEditor.exe`.**
3. **Кайфовать от члена в жопе.** *(Классический экспириенс линуксоида).*
